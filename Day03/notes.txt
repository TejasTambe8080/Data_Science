🧠 PART 1 — FUNCTIONS (DEEP + INTERVIEW LEVEL)
🔹 What is a Function?

👉 Reusable block of code

def add(a, b):
    return a + b
🔹 Types of Functions
# Default argument
def greet(name="User"):
    return "Hello " + name

# Keyword arguments
greet(name="Tejas")
🔹 Why Functions Matter (Industry)
Reusability
Clean code
Easier debugging
Used in pipelines
⚡ PART 2 — LAMBDA FUNCTIONS (IMPORTANT)
square = lambda x: x * x
print(square(5))

👉 Used for:

Short operations
Inline transformations
⚡ PART 3 — MAP & FILTER (INTERVIEW FAVORITE)
🔹 map()
nums = [1,2,3,4]

squares = list(map(lambda x: x*x, nums))
print(squares)
🔹 filter()
nums = [10, 15, 20, 25]

evens = list(filter(lambda x: x % 2 == 0, nums))
print(evens)
🔹 Industry Use
Data transformation
Preprocessing pipelines
⚠️ PART 4 — EXCEPTION HANDLING (VERY IMPORTANT)

👉 Real systems NEVER crash

try:
    x = int("abc")
except ValueError:
    print("Invalid input")
🔹 Multiple Exceptions
try:
    a = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")
🔹 Finally Block
finally:
    print("Execution done")
🏗️ PART 5 — MODULAR CODE (INDUSTRY LEVEL)

from utils import clean_text

🔹 Why modular?
Scalable projects
Team collaboration
Clean architecture
