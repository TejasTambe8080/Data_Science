👉 Built on top of Matplotlib

🔹 Why Seaborn?
Better design
Less code
Built-in statistical plots
📊 PART 2 — IMPORTANT PLOTS (MUST KNOW)
🔹 1. Histogram + KDE (Distribution)
import seaborn as sns
import matplotlib.pyplot as plt

data = [10,20,20,30,30,30]

sns.histplot(data, kde=True)
plt.show()

👉 Shows:

Distribution
Density
🔹 2. Box Plot (Outliers Detection)
sns.boxplot(x=data)
plt.show()

👉 Used for:

Finding outliers
🔹 3. Count Plot (Category Count)
dept = ["IT", "HR", "IT", "HR", "IT"]

sns.countplot(x=dept)
plt.show()
🔹 4. Heatmap (VERY IMPORTANT)
import pandas as pd

df = pd.DataFrame({
    "a": [1,2,3],
    "b": [4,5,6]
})

sns.heatmap(df.corr(), annot=True)
plt.show()

👉 Shows:

Correlation between variables
🔹 5. Scatter Plot (Relationship)
x = [1,2,3]
y = [10,20,30]

sns.scatterplot(x=x, y=y)
plt.show()
