💻 PART 2 — PIVOT TABLE (CORE)
import pandas as pd

data = {
    "dept": ["IT", "HR", "IT", "HR"],
    "salary": [50000, 40000, 60000, 45000]
}

df = pd.DataFrame(data)

pivot = pd.pivot_table(df, values="salary", index="dept", aggfunc="mean")

print(pivot)
🔹 Multiple Aggregations
pd.pivot_table(df, values="salary", index="dept", aggfunc=["mean", "max", "min"])
📊 PART 3 — MULTI-LEVEL GROUPING
data = {
    "dept": ["IT", "IT", "HR", "HR"],
    "role": ["Dev", "Manager", "Dev", "Manager"],
    "salary": [50000, 80000, 40000, 70000]
}

df = pd.DataFrame(data)

df.groupby(["dept", "role"])["salary"].mean()

👉 Used in:

Deep analysis
Hierarchical data
