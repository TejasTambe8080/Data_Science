👉 Pattern matching tool for text

Extract emails

Clean unwanted text
🔹 Why Important in Data Science?
Data cleaning (logs, reviews, chat data)
NLP preprocessing
Feature extraction
🧠 PART 2 — BASIC REGEX PATTERNS
Pattern	Meaning	Example
\d	Digit	123
\w	Word	abc
.	Any char	a,1,@
+	One or more	aaaa
*	Zero or more	optional
^	Start	start match
$	End	end match
💻 PART 3 — USING REGEX IN PYTHON
import re

text = "My number is 9876543210"

nums = re.findall(r"\d+", text)
print(nums)
🔹 Extract Emails
text = "Contact: test@gmail.com"

emails = re.findall(r"\S+@\S+", text)
print(emails)
🔹 Replace Text
text = "Hello 123"

clean = re.sub(r"\d+", "", text)
print(clean)
