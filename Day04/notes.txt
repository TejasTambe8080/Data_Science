👉 Reading/writing data from files (CSV, TXT, JSON)

👉 In industry:

First step = load data correctly
🔹 File Modes
Mode	Meaning
"r"	Read
"w"	Write (overwrite)
"a"	Append
🔹 Basic File Read
f = open("data.txt", "r")
print(f.read())
f.close()
🔹 Best Practice (IMPORTANT)
with open("data.txt", "r") as f:
    data = f.read()

👉 Why with?

Automatically closes file
Prevents memory issues
💻 PART 2 — TEXT FILE HANDLING
Write file
with open("sample.txt", "w") as f:
    f.write("Hello Data Science")
Read line-by-line
with open("sample.txt", "r") as f:
    for line in f:
        print(line.strip())
📊 PART 3 — CSV HANDLING (IMPORTANT)
🔹 Using Python (basic)
import csv

with open("data.csv", "r") as f:
    reader = csv.reader(f)
    for row in reader:
        print(row)
🔹 Using Pandas (INDUSTRY WAY)
import pandas as pd

df = pd.read_csv("data.csv")

print(df.head())

👉 This is what companies use.

⚠️ PART 4 — ERROR HANDLING WITH FILES
try:
    with open("data.csv", "r") as f:
        print(f.read())
except FileNotFoundError:
    print("File not found")
