🧠 PART 1 — THEORY (INTERVIEW LEVEL)
🔹 What is Pandas?

👉 Library used for data manipulation & analysis

🔹 Key Structures
✅ Series

👉 1D data

a = pd.Series([10, 20, 30])
✅ DataFrame

👉 Table (rows + columns)

data = {
    "name": ["A", "B"],
    "marks": [80, 90]
}
df = pd.DataFrame(data)
🔹 Why Pandas?
Handles large datasets
Easy filtering & transformation
Industry standard
