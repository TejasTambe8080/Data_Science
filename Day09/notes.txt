🔹 What is Datetime?

👉 Represents date + time (YYYY-MM-DD HH:MM:SS)

👉 Used in:

Sales analysis
User activity tracking
Logs
🔹 Why Important in Data Science?
Trend analysis
Time-based features
Forecasting
💻 PART 2 — PYTHON DATETIME MODULE
from datetime import datetime

now = datetime.now()
print(now)
🔹 Convert String → Date
date = datetime.strptime("2026-04-25", "%Y-%m-%d")
print(date)
🔹 Extract Components
print(date.year)
print(date.month)
print(date.day)
📊 PART 3 — PANDAS DATETIME (INDUSTRY WAY)
import pandas as pd

data = {
    "date": ["2024-01-01", "2024-02-15"]
}

df = pd.DataFrame(data)

df["date"] = pd.to_datetime(df["date"])

print(df["date"].dt.year)
print(df["date"].dt.month)
🔹 Add New Features
df["day"] = df["date"].dt.day
df["weekday"] = df["date"].dt.day_name()
