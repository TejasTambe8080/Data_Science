🧠 PART 1 — THEORY (INTERVIEW LEVEL)
🔹 What is NumPy?

👉 Library for fast numerical operations

🔹 Why NumPy?
Faster than Python lists
Supports vector operations
Used in ML libraries
🔹 List vs NumPy Array
Feature	List	NumPy Array
Speed	Slow	Fast
Math Ops	Limited	Powerful
Use	General	Data Science
💻 PART 2 — NUMPY BASICS
import numpy as np

arr = np.array([1, 2, 3, 4])
print(arr)
🔹 Create Arrays
np.zeros(5)
np.ones(5)
np.arange(1, 10)
🔹 Indexing & Slicing
arr = np.array([10, 20, 30, 40])

print(arr[1])
print(arr[1:3])
⚡ PART 3 — VECTOR OPERATIONS (VERY IMPORTANT)
arr = np.array([1, 2, 3])

print(arr + 5)
print(arr * 2)

👉 No loops needed → faster

🔹 Math Functions
np.mean(arr)
np.sum(arr)
np.max(arr)
np.min(arr)
📊 PART 4 — 2D ARRAYS (IMPORTANT)
arr = np.array([[1,2,3], [4,5,6]])

print(arr.shape)
print(arr[0][1])
